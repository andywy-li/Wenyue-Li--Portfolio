# Wenyue Li – Projects Portfolio

This repository collects projects that Wenyue Li completed either independently or as a primary contributor.

## Repository Overview

- **Course Projects**  
  Each course project is organized in a folder titled with the course number (including both EECS and ME courses).  
  Inside each course folder, there are:
  - The main project code and/or report.
  - A short description of the course and the context of the project (course name, learning goals and techniques used).

- **Teaching Assistant Materials**  
  This repository also includes materials created while Wenyue Li was serving as a Teaching Assistant, such as:
  - Review sheets and practice problems.

- **Research Projects**  
  In addition, there are folders for research projects where Wenyue Li was a main member of the team.  
  These folders may contain:
  - Project summaries and papers or drafts.
  - Experimental code, simulations, and data processing scripts.

Overall, this repository provides a consolidated view of Wenyue Li’s technical work, teaching experience, and research involvement.
